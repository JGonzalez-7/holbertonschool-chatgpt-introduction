# Debugging and Automation Exercises

## Objective
- Debugging: Use ChatGPT to diagnose and correct bugs in provided code samples across multiple languages, sharpening your problem-solving skills.
- Automation: Automate repetitive tasks such as generating boilerplate code, documentation, and basic unit tests so you can focus on higher-value work.

## Outcomes
- Enhanced debugging skills, with clear articulation of problems and AI-assisted fixes.
- Automation proficiency that leads to more structured and error-free code.
